Artifacts class 
: στην κλάση αυτή έχω δημιουργήσει τα ζητούμενα πεδία όπως και τις δυο μεθόδους συναντήσεων getIndex,getInfo.φυσικά μέσω contractor δημιουργώ ένα δημιούργημα artifact στο οποίο περνώ μέσω πρισμάτων τις τιμές  που χρειάζεται.Αυτή η κλάση είναι abstract γιατί δεν πμ μπορούν να δημιουργηθούν αντικείμενα και οι μέθοδοι της θεωρούνται abstract καθώς είναι μια κλάση που έχει περισσότερο κατασκευαστικούς σκοπούς.Υπάρχει και η μέθοδος evaluate() για κατασκευαστικούς λόγους χωρίς κάποια χρήση στην παρούσα κλάση.

Masterpiece class 
: η κλάση αυτή όπως και η προηγούμενη απαρτίζεται από ορισμένα fields από τον constructor της και την μεθόδο getInfo η οποία με χρήση των μεθόδων τις Artifact εκτυπώνει τα ζητούμενα στοιχεία παίρνοντας ότι χρειάζεται από την άλλη κλάση μέσω super για πρόσβαση σε μέλη υπέρκλάσης.Έχω επιλέξει να έχω και την παρόν κλάση abstract καθώς  και την μέθοδο της evaluate() η οποία υπάρχει για κατασκευαστικούς σκοπούς χωρίς κάποια υλοποίηση στην παρούσα κλάση .

Στην συνέχεια υπάρχουν δυο  κλάσεις που κληρονομούν την Masterpiece οι οποίες είναι οι Paintings Sculptures οι οποίες δημιουργούν τα αντικείμενα που θα ασχοληθούμε και περιέχουν περίπου ίδια πράγματα
